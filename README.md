# MERN Blog Platform

A full-stack blogging platform built with the MERN stack, designed to provide a secure and seamless blogging experience. The application enables users to create, manage, and interact with blog posts through a modern React frontend and a scalable Node.js backend.

**React • Node.js • Express.js • MongoDB • JWT Authentication**

---

## Overview

The MERN Blog Platform is a production-inspired web application that demonstrates the development of a complete full-stack system using modern JavaScript technologies.

It allows users to register, authenticate securely, publish blog posts, edit or delete their own content, and participate in discussions through comments. The project follows RESTful API principles and adopts a structured architecture to ensure maintainability and scalability.

This project was developed to strengthen my understanding of full-stack development, authentication, API design, database management, and frontend-backend integration while following industry-standard development practices.

---

## Key Features

- Secure user authentication using JWT
- Password hashing with bcrypt
- Create, edit, delete, and view blog posts
- Comment system for user interaction
- Responsive React frontend
- RESTful API architecture
- MongoDB database integration
- Protected routes and authorization
- Modular backend architecture
- Error handling and input validation
